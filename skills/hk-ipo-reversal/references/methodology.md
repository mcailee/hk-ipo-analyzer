# 方法论：港股新股暗盘反转猎手 V2 (期望偏差版)

## 核心概念：期望偏差 (Expectation Gap)

### 定义

```
deviation = dark_return - expected_return
```

- `dark_return`: 暗盘实际涨跌幅 (%)
- `expected_return`: 基于历史数据推算的合理预期涨幅 (%)
- `deviation`: 期望偏差 (%)。负值表示不及预期，正值表示超预期。

### 预期涨幅推算方法

基于全量128只历史数据集的分档统计映射：

1. **超购区间基准** (权重60%)
   - 按超购倍数分6档 (<20x, 20-100x, 100-500x, 500-2000x, 2000-5000x, >5000x)
   - 计算各档的历史首日平均涨幅作为基准

2. **基石修正** (权重一部分)
   - 有基石投资者 → 修正 = 有基石组均值 - 全局均值
   - 无基石 → 修正 = 无基石组均值 - 全局均值

3. **行业修正** (权重一部分)
   - 目标编码：各行业首日涨幅均值 - 全局均值

4. **募资规模修正** (权重一部分)
   - ≥100亿: -12pp (超大盘折价)
   - ≥50亿: -8pp
   - ≥20亿: -3pp
   - ≥10亿: 0
   - ≥5亿: +2pp
   - <5亿: +5pp (小盘溢价)

5. **综合**: expected = baseline × 0.6 + (baseline + 各修正) × 0.4

## 三类分析对象

| 类型 | 条件 | 偏差特征 | 典型案例 |
|------|------|----------|----------|
| 暗盘下跌 | dark_return < 0 | 必然大负 | 埃斯顿(暗盘-7.5%, 偏差-36%) |
| 涨幅不及预期 | dark_return ≥ 0, deviation ≤ -10 | 中等负值 | 华沿机器人(暗盘+5%, 偏差-25%) |
| 潜力释放 | dark > 0, day10-dark > 15pp | 偏差可能负 | 经发物业(暗盘+8%, day10+30%) |

## 自动偏差阈值寻优

扫描参数空间：
- **偏差阈值**: [-5, -10, -15, -20, -25, -30, -40, -50]
- **回正窗口**: Day1, Day3, Day5, Day7, Day10
- **回正标准**: 回正(>0%)

综合评分 = 修正率 × log(修正组均收益+1) × 样本量权重

## 多因子修正模型

### 因子列表 (8个)

| 编号 | 因子 | 说明 |
|------|------|------|
| F1 | 超购倍数(log) | 市场热度 |
| F2 | 基石投资者 | 机构背书 |
| F3 | 募资规模(log) | 流动性和关注度 |
| F4 | 期望偏差 | 核心因子：不及预期的程度 |
| F5 | 偏差绝对值 | 偏离程度 |
| F6 | 暗盘波幅 | 暗盘交易活跃度 |
| F7 | 首日换手率 | 首日交易活跃度 |
| F8 | 暗盘涨跌幅 | 暗盘绝对涨跌 |

### 训练方法

- 手写 Logistic 回归 (SGD, L2正则化)
- 特征标准化 (z-score)
- 2000轮迭代, 学习率0.1, 正则化λ=0.1
- 双轨验证：Logistic权重 50% + 信息增益 50%

## 潜力释放分析

筛选条件：暗盘上涨 (dark > 0) 且后续大幅补涨 (max(day5, day7, day10) - dark > 15pp)

分析内容：
- 共性特征统计（平均超购、基石比例、行业分布）
- 价格路径展示
- 补涨幅度排序

## 价格路径聚类

- 方法：层次聚类 (Ward 方法, 欧式距离)
- 对象：所有偏差为负的股票 (deviation < 0)，不限暗盘必须下跌
- 聚类数：4 (V型修正 / U型缓慢修正 / L型持续低迷 / 断崖式)
- 按最终收益排序分配标签

| 对比项 | V1 | V2 |

| 分析维度 | 暗盘涨跌幅 | 期望偏差 |
| 入选条件 | 暗盘 < 0 | deviation < 阈值 |
| 核心样本 | 27只 | 51只 |
| 广义样本 | 33只 | 72只 |
