# 港股 IPO 打新分析 — 11 维度评分方法论

## 评分总则

- 每个维度独立评分 0-100 分
- 数据缺失时采用中性分 50 分，并标注"数据不足"
- 各维度按配置权重加权求和得出综合评分
- Phase1（招股期）仅用 9 个维度，权重等比放大至 100%
- Phase2（认购期）使用全部 11 个维度

## 各维度评分标准

### 1. 估值定价（18%）

| 子指标 | 评分逻辑 |
|--------|----------|
| PE同行折价率 | 折价>30%→90, 10-30%→75, 0-10%→60, 溢价0-20%→40, >20%→20 |
| 定价区间位置 | 下限(0-30%)→85, 中间→65, 偏上限→45, 上限→25 |
| 可比IPO首日表现 | 涨幅>20%→85, >5%→70, >0%→55, <0%→30 |
| 市值规模 | <20亿→75, <100亿→65, >100亿→55 |

### 2. 市场认购热度（15%，Phase2）

| 子指标 | 评分逻辑 |
|--------|----------|
| 公开认购倍数 | >500x→95, >100x→85, >30x→70, >10x→55, >1x→40, <1x→15 |
| 国际配售倍数 | >20x→90, >10x→80, >5x→65, >2x→50, <2x→30 |
| 回拨触发 | 触发→80, 未触发→45 |
| 同期竞争 | ≤2只→80, ≤5只→60, >5只→35 |
| 近期破发率 | <20%→80, <40%→60, <60%→40, >60%→20 |

### 3. 财务状况（14%）

| 子指标 | 评分逻辑 |
|--------|----------|
| 营收CAGR | >50%→95, >30%→85, >15%→70, >5%→55, 0-5%→40, <0%→20 |
| 净利润率 | >20%→90, >10%→75, >5%→60, 0-5%→45, <0%→25 |
| 毛利率 | >60%→90, >40%→75, >25%→55, <25%→35 |
| 资产负债率 | <30%→90, <50%→75, <65%→55, <80%→35, >80%→15 |
| ROE | >20%→90, >15%→80, >10%→65, >5%→50, <5%→30 |

### 4. 行业竞争（10%）

基于行业分类关键词匹配景气度，热门赛道（AI/半导体/SaaS等）85分，低迷行业30分，其余60分。并叠加行业专属指标。

### 5. 基石投资者（10%）

- 无基石: 35分
- 数量: ≥5家→80, ≥3→70, ≥1→55
- 质量: 顶级机构+15, 关联方>50%扣20分
- 占比: >70%扣15分（负面信号）
- 锁定期: ≥12月→85, ≥6月→70, <6月→45

### 6. 公司基本面（8%）

成立年限、业务清晰度、员工规模简单评估。

### 7. 承销发行（8%）

保荐人资质（知名行排名）、承销团规模、历史破发率、递表次数。

### 8. 上市后流动性（7%）

自由流通市值、港股通可能性、做市商、日均成交预估。

### 9. 绿鞋机制（5%）

有/无、超额配售比例、稳价期天数。

### 10. 股东构成（3%）

实控人持股合理性、同股不同权、代持风险。

### 11. 法律诉讼（2% + 降级）

涉诉数量/金额、刑事诉讼、监管调查。红线触发评级降一档。

## 四级评级

| 评级 | 分数区间 |
|------|----------|
| 强烈推荐 | ≥ 80 |
| 推荐 | 65-79 |
| 中性 | 50-64 |
| 回避 | < 50 |
