# 港股 IPO 打新分析 — 15 维度评分方法论 (v3.0)

## 版本变更 (v3.0)

- **扩展至 15 维度**：12 个标准维度 + 3 个条件性维度（同批次对比、A+H 股、暗盘数据）
- **权重归一化**：12 个标准维度权重 = 100%，3 个条件维度额外 +15%。当条件维度参与时，scorer 自动归一化至 100%（例如全部 15 维度可用时，估值有效权重 = 15/115 ≈ 13.0%）
- **新增「市场情绪」维度**（权重 9%）：恒指表现、破发率、南向资金、波动率
- **权重实证校准**：基于回测数据重新分配 12 个维度权重
- **认购倍数对数化**：解决 1x~11465x 跨度下线性阈值失真问题
- **机制A/B适配**：兼容 2025.08 港交所 IPO 定价机制改革
- **发行规模独立评估**：微型IPO炒作警告，大型IPO机构参与度评估
- **卖出时机建议模型**：基于市值/认购倍数/评分的退出策略
- **概率预测区间**：连续线性插值，消除阶梯函数边界跳变
- **中签率计算器**：基于发行价推断每手股数

## 评分总则

- 每个维度独立评分 0-100 分
- 数据缺失时采用中性分 50 分，并标注"数据不足"
- 各维度按配置权重加权求和得出综合评分，总权重自动归一化至 100%
- Phase1（招股期）使用 10 个标准维度（含市场情绪），权重等比放大至 100%
- Phase2（认购期）使用全部 12 个标准维度 + 可用的条件维度，输出卖出时机建议
- 条件维度（同批次对比/A+H股/暗盘数据）仅在数据可用时参与评分

## 各维度权重（回测校准后）

| 维度 | 权重 | 阶段 | v1 权重 | 调整理由 |
|------|------|------|---------|----------|
| 认购热度 | 16% | P2 | 15% | 回测显示其为首日涨幅最强预测因子 |
| 估值定价 | 15% | P1 | 18% | 略降，部分预测力被情绪维度分摊 |
| 财务状况 | 12% | P1 | 14% | 对打新首日回报解释力有限 |
| 行业竞争 | 12% | P1 | 10% | 行业赛道景气度影响显著 |
| 基石投资 | 10% | P1 | 10% | 保持不变 |
| 市场情绪 | 9%  | P1 | 新增 | 回测中系统性因子，解释 ~9% 变异 |
| 承销发行 | 8%  | P1 | 8%  | 保持不变 |
| 上市流动 | 7%  | P2 | 7%  | 保持不变 |
| 公司面 | 6%  | P1 | 8%  | 对短期表现预测力较弱 |
| 绿鞋机制 | 3%  | P1 | 5%  | 护盘效果有限 |
| 股东构成 | 1%  | P1 | 3%  | 短期影响极小 |
| 法律诉讼 | 1%+降级 | P1 | 2% | 保留降级机制，权重降低 |

## 各维度评分标准

### 1. 估值定价（15%）

| 子指标 | 评分逻辑 |
|--------|----------|
| PE同行折价率 | 折价>30%→90, 10-30%→75, 0-10%→60, 溢价0-20%→40, >20%→20 |
| 定价区间位置 | 下限(0-30%)→85, 中间→65, 偏上限→45, 上限→25 |
| 可比IPO首日表现 | 涨幅>20%→85, >5%→70, >0%→55, <0%→30 |
| 发行规模评估 | 微型(<200M)→55⚠️, 小型→70, 中型→68, 大型→65, 超大型→60 |

### 2. 市场认购热度（16%，Phase2）

**机制A（传统）:**

| 子指标 | 评分逻辑 |
|--------|----------|
| 公开认购倍数 | **对数化**: score = 20 + 10.5 × ln(mult)，上限98 |
| 国际配售倍数 | >20x→90, >10x→80, >5x→65, >2x→50, <2x→30 |
| 回拨触发 | 触发→80, 未触发→45 |
| 中签率 | ≤1%→90, ≤5%→78, ≤15%→65, ≤30%→50, ≤60%→35, >60%→18 |

**机制B（2025.08改革后）:**

| 子指标 | 评分逻辑 |
|--------|----------|
| 机构下单倍数 | >10x→95, >5x→80, >2x→65, >1x→50, <1x→25 |
| 定价位置 | 下限→85, 中间→70, 偏上→50, 上限→30 |
| 散户意向倍数 | 对数化（同公开认购倍数公式） |

**通用子指标（A/B共享）:**
| 同期竞争 | ≤2只→80, ≤5只→60, >5只→35 |
| 近期破发率 | <20%→80, <40%→60, <60%→40, >60%→20 |

### 3. 市场情绪（9%，新增）

| 子指标 | 评分逻辑 |
|--------|----------|
| 恒指近1月表现 | >5%→85, >2%→75, >0%→60, >-3%→45, <-3%→25 |
| 近30日破发率 | <15%→90, <30%→70, <50%→50, <70%→30, >70%→15 |
| 南向资金净流入 | >50亿→85, >20亿→72, >0→58, >-30亿→40, <-30亿→22 |
| 恒指波动率 | <15%→80, <20%→65, <25%→50, <35%→35, >35%→18 |

### 4. 财务状况（12%）

| 子指标 | 评分逻辑 |
|--------|----------|
| 营收CAGR | >50%→95, >30%→85, >15%→70, >5%→55, 0-5%→40, <0%→20 |
| 净利润率 | >20%→90, >10%→75, >5%→60, 0-5%→45, <0%→25 |
| 毛利率 | >60%→90, >40%→75, >25%→55, <25%→35 |
| 资产负债率 | <30%→90, <50%→75, <65%→55, <80%→35, >80%→15 |
| ROE | >20%→90, >15%→80, >10%→65, >5%→50, <5%→30 |

### 5. 行业竞争（12%）

基于行业分类关键词匹配景气度，热门赛道（AI/半导体/SaaS等）85分，低迷行业30分，其余60分。并叠加行业专属指标。

### 6. 基石投资者（10%）

- 无基石: 35分
- 数量: ≥5家→80, ≥3→70, ≥1→55
- 质量: 顶级机构+15, 关联方>50%扣20分
- 占比: >70%扣15分（负面信号）
- 锁定期: ≥12月→85, ≥6月→70, <6月→45

### 7. 承销发行（8%）

保荐人资质（知名行排名）、承销团规模、历史破发率、递表次数。

### 8. 上市后流动性（7%）

自由流通市值、港股通可能性、做市商、日均成交预估。

### 9. 公司基本面（6%）

成立年限、业务清晰度、员工规模简单评估。

### 10. 绿鞋机制（3%）

有/无、超额配售比例、稳价期天数。

### 11. 股东构成（1%）

实控人持股合理性、同股不同权、代持风险。

### 12. 法律诉讼（1% + 降级）

涉诉数量/金额、刑事诉讼、监管调查。红线触发评级降一档。

## 条件性维度（数据可用时参与评分）

### 13. 同批次对比（5%，Phase2，条件性）

| 子指标 | 评分逻辑 |
|--------|----------|
| 认购倍数排名 | 同批次百分位排名 → top20→90, top40→75, mid→60, bottom40→45, bottom20→30 |
| PE估值排名 | 同批次PE排名（越低越好），同上映射 |
| 同批次破发率 | <15%→90, <30%→70, <50%→50, <70%→30, >70%→15 |
| 同批次首日表现 | 平均涨幅 >20%→90, >10%→78, >5%→65, >0%→50, >-5%→38, <-5%→20 |
| 发行规模对比 | 与同批次均值比较，接近→70, 偏小→65/55, 偏大→60 |

### 14. A+H 股分析（4%，Phase2，条件性）

| 子指标 | 评分逻辑 |
|--------|----------|
| A/H 溢价率 | A股大幅溢价(>40%)→90, 中等→80/68, 持平→50, H股反而贵→35/20 |
| H/A PE折让 | H股PE大幅低于A股→90/75, 接近→60, H股反而贵→45/25 |

### 15. 暗盘数据（6%，Phase2，条件性）

| 子指标 | 评分逻辑 |
|--------|----------|
| 暗盘溢价率 | >20%→95, >10%→85, >5%→75, >0%→60, >-3%→42, >-10%→25, <-10%→10 |

暗盘数据是首日涨幅最强的短期预测信号，也用于概率预测模型中的修正。

## 卖出时机建议模型（Phase2 专属）

| 条件 | 策略 | 持有期 | 止损/止盈 |
|------|------|--------|-----------|
| 小市值(<50亿) + 高认购(>30x) | ⚡ 速战速决 | T+1~T+3 | -5% / +15% |
| 大市值(>300亿) + 高评分(>75) | 🏦 中线持有 | 1-3月 | -10% / +40% |
| 超高认购(>100x) | ⚡ 速战速决 | T+1~T+3 | -5% / +15% |
| 中等评分(>70) | 📈 短线持有 | 1-2周 | -8% / +25% |
| 其他 | ⚡ 速战速决 | T+1~T+3 | -5% / +15% |
